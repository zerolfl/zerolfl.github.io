Number 数字

主要有四种类型：int（整型）、float（浮点数）、complex（复数）、bool（布尔值）。

运算：+、-、*、/、//（整除）、%（取余）。

常用函数：abs(n)（取绝对值）、max()、min()、power(x,y)（xyx^yxy）、sqrt()（不属于Python的内置函数，需要import math）。

List 列表

用[]括起来的数据集。

索引是从0开始的。

里面的元素可以是相同类型或不同类型的元素。

以a = [1, 2, 3, 4, 5]为例，List中涉及到的几个主要操作：

  1. 正查：a[index]，index指索引；

  2. 增加：a.append(item)，item指要在末尾加入的新元素；

  3. 更新：a[index] = new_data；

  4. 删除：a.pop()（删除末尾元素，返回值为这个被删除的元素值），a.pop(index)（删除指定index位置的元素，返回删除的元素值）；

  5. 判断是否存在某个元素：value in a，判断列表a中是否有value，返回bool值；

  6. 切片slice操作：a[0:3]展示索引从0到2的元素。

    x:y：左闭右开区间，即[x,y)[x,y)[x,y)；

    a[1:-1]：-1表示最后一个元素的索引，但是这个slice取不到最后一个元素；

    a[1:]：:之后没写，默认是从指定索引取到最后一个元素；

    a[:]：:前后都没有值，默认是取所有元素。

常用函数：

  1. len(a)：返回列表a的长度；
  2. max(a)：取最大
  3. min(a)：取最小
  4. a.reverse()：a的元素颠倒排序；
  5. a.clear()：清除a中所有元素。

遍历/迭代List：

  1. # 直接遍历列表的元素
for i in a:
    print(i)

Tuple 元组

用()括起来的数据集。

里面的元素可以是相同类型或不同类型的元素。

List与Tuple的区别：

  • List支持：增、删、改、查；
  • Tuple支持：查（即，无法修改集合中的内容）。

Tuple主要用在查的情况！避免误操作修改了集合中的内容。

以a = (1, 2, 3, 4, 5)为例，Tuple的查找：

  1. 正查：a[index]，返回索引为index的元素值；
  2. 反查：a.index(value)，返回value的索引。

常用函数：

  1. len(a)：返回元组a的长度；
  2. max(a)：取最大
  3. min(a)：取最小

常用操作（与List一样）：

  1. 判断是否存在某个元素：value in a，判断列表a中是否有value，返回bool值；
  2. 切片slice操作；
  3. 遍历/迭代操作。

注意：

  • 当a = (1)时，a是int类型；
  • 当a = (1,)时，a是tuple类型；

Set 集合

用{}括起来的数据集，且其中所有的元素是唯一的。

set中的元素是无序的，即不是按照索引来获取对应值的。

理解成：圈养了独一无二的一堆数。这种结构比较非主流。

以a = {1, 2, 3, 4, 5}为例，常用操作：

  1. 创建集合：a = set()。

  2. 增加元素：a.add(item)，添加一个元素item；

    如果set中已经有item，那么改set不会变。

  3. 增加元素：a.update(iteratable_object)，加上一堆可迭代数据类型中的元素值。；

  4. 删除元素：a.remove(item)，删除item元素；

    在set中，用a.pop()没啥意义，因为set无索引。

  5. 判断是否存在某个元素：item in a，返回bool值。

常用运算：

  1. -：如a - b指a有b没有的元素（差集）；
  2. |：如a | b指a有或b有的元素（并集）；
  3. &：如a & b指a有且b有的元素（交集）；
  4. ^：如a ^ b指a和b中不同时有的元素（并集减交集）

常见的set：HashSet（比较常见）、LinkListSet、TreeSet。

Dictionary 字典

用{}括起来，元素为键-值对key:value。

如a = {"name":"learning", "age":18}。

查用操作：

  1. 正查：a[key]，利用key来查；

    相当于给set加了个索引信息，这个索引信息就是key。

  2. 增加（修改）元素：a[key] = value，增加（修改）索引为key，值为value的键值对；

  3. 删除元素：a.pop(key)，删除索引为key，值为value的键值对；

  4. 判断是否存在某个键key：key in a，返回bool值。

    注：字典的in操作是判断是否存在指定的”字“（键key），而不是判断是否存在指定的值（value）；

String 字符串

用""引起来的内容，其中每个元素为字符。即，由一个个字符所组成的数组。

以a = "hello World"为例，常用操作：

  1. 正查、切片操作

  2. a.count(str)：计数item字符出现了多少次；

  3. a.isupper()：判断a中的字母是否全为大写字母，返回bool值；

  4. a.islower()：判断a中的字母是否全为小写字母，返回bool值；

  5. a.isdigit()：判断a中的字符是否全为数字字符，返回bool值；

  6. a.upper()：返回将a中的字母改为大写的结果，a没有被修改；

  7. a.lower()：返回将a中的字母改为小写的结果，a没有被修改；

  8. a.strip()：返回将a中最前最后的空格删除的结果，a没有被修改；

  9. a.lstrip()：返回将a中最前的空格删除的结果，a没有被修改；

  10. a.rstrip()：返回将a中最后的空格删除的结果，a没有被修改；

  11. a.swapcase()：返回将a中的字母大小写互换的结果，a没有被修改；

  12. a.replace(old, new)：返回将a中的old字符（串）替换为new字符（串），a没有被修改；

  13. a.split(seq)：返回根据分隔规则seq将字符串a进行分割的结果，a没有被修改。

    a.split()：默认是以空格为规则来分隔，即a.split(" ")；

常用函数：

  1. len(a)：返回a的长度；
  2. max(a)：取ascii码最大的字符
  3. min(a)：取ascii码最小的字符

Array 数组

其元素类型相同，且需要用一块连续的内存空间来存储的数据集。

可以用List来表示Array，要求List中的元素类型相同。

优点：正查（访问，access）的时间复杂度为O(1)O(1)O(1)；

缺点：反查（search）、插入和删除的时间复杂度为O(n)O(n)O(n)。

nnn表示数组长度。

适合读，不适合写。

LinkedList 链表

链表的元素是节点，每个节点包含两个信息（值和指向下一个Node的信息）。

数组和链表的区别：

优点：插入（仅针对插入这个操作）和删除的时间复杂度为O(1)O(1)O(1)；

缺点：正查和反查的时间复杂度为O(n)O(n)O(n)。

适合写，不适合读。

有单端、双端链表。

Python的deque就是一种链表。

以其为例，常用操作如下：

  1. 创建链表：linkedlist = deque()

    需要from collections import deque

  2. 添加元素item：linkedlist.append(item)、linkedlist.insert(pos, item)

    注意这里的insert()时间复杂度是O(n)O(n)O(n)，因为这是实现插入的函数，还包括了查找到这个pos的时间。

  3. 访问第index个元素：linkedlist[index]

    时间复杂度为O(n)O(n)O(n)。

  4. 更新第index个元素：linkedlist[index] = new_data

    时间复杂度为O(n)O(n)O(n)。

  5. 查找元素item：linkedlist.index(item)

    时间复杂度为O(n)O(n)O(n)。

  6. 删除元素item：linkedlist.remove(item)

    删除指定索引index上的元素则为：del linkedlist[index]；

    时间复杂度也是O(n)O(n)O(n)，还包括了查找到这个元素的时间。

  7. 链表的长度：len(linkedlist)

    时间复杂度为O(1)O(1)O(1)。

Hash Table 哈希表

Python的dictionary就是一种哈希表（散列表）。

通过哈希算法将key映射成一个唯一一个索引，来帮助访问对应的value。

优点：正查、插入和删除的时间复杂度为O(1)O(1)O(1)；

读写都OK。

哈希碰撞：2个不同的key通过一个哈希函数得到相同的内存地址。解决方法：例如可以使用链表来解决。

Queue 队列

队列Queue：队列仅有一个入口和一个出口。

双端队列Deque：数据项既可以从队首加入，也可以从队尾加入；数据项也可以从两端移除。

先入先出（First-In First-Out, FIFO）；“行为绅士”

可以用List来实现；

主要用在BFS中。

Python的话可以使用deque。具体操作如下（也可以作为单端队列使用）：

  1. 创建链表：linkedlist = deque()

    需要from collections import deque

    注意这里的insert()时间复杂度是O(n)O(n)O(n)，因为这是实现插入的函数，还包括了查找到这个pos的时间。

    时间复杂度为O(n)O(n)O(n)。

  4. 获取即将出队的元素：queue[0]。

    时间复杂度为O(1)O(1)O(1)。

Stack 栈

数据项的加入和移除都仅发生在同一端（栈顶）。

后入先出（Last-In First Out，LIFO）；如浏览器后退功能。

可以用List来实现；

主要用在DFS中。

可以直接用Python的List来实现栈的一些操作。

Heep 堆

一种完全二叉树的结构，且每个节点≥\geq≥子节点（最大堆），或每个节点≤\leq≤子节点（最小堆）。

最大堆：堆顶是最大值，每次pop()都pop掉这组数的最大值（堆顶元素）；

最小堆：堆顶是最小值，每次pop()都pop掉这组数的最小值（堆顶元素）。

heapq的包不支持大堆，如果要用大堆，那就把数组都乘−1-1−1，然后用小堆来实现。

访问：无，一般不通过索引来访问堆中元素；

搜索：指只查看堆顶元素O(1)O(1)O(1)

添加：O(log⁡n)O(\log n)O(logn)

删除：O(log⁡n)O(\log n)O(logn)

堆化：一堆无序的数→\rightarrow→完全二叉树→\rightarrow→最大堆/最小堆

第一个箭头：O(n)O(n)O(n)，只要遍历一次数组即可构建；

第二个箭头：O(n)O(n)O(n)，最差的情况就是每层元素需要一直往下去交换，相当于每层需要操作数=每层元素*单元素逐渐往下交换的次数（从该层走下最底层）。

Tree 树

连接的节点与节点之间有父子关系。

节点：

  • 根节点：最祖先的那个节点；
  • 叶子节点：无下一个子节点。

其他概念：

  • 高度：节点到叶子节点的最长路径；

    树的高度：根节点到叶子节点的最长路径。

  • 深度：根节点到节点的边数；

  • 层：同辈节点为一层，最祖先的节点为第一层。

二叉树：

  1. 普通二叉树：每个节点最多有两个子节点；

  2. 满二叉树：除了叶子节点，每个节点都有两个子节点（左右节点）；且每个叶子都在同一层；

  3. 完全二叉树：对树中的节点从上至下、从左至右编号，这些节点的编号与满二叉树节点的位置相同。（从上至下、从左至右依次填满）

    想象满二叉树的右侧方缺少右节点的样子。

二叉树的遍历：

  1. 前序遍历：根节点→\rightarrow→左子树→\rightarrow→右子树
  2. 中序遍历：左子树→\rightarrow→根节点→\rightarrow→右子树
  3. 后续遍历：左子树→\rightarrow→右子树→\rightarrow→根节点

这个”序“看”根节点“来确定。

遍历示意如下：

  1. 前序遍历：A→B→D→E→C→F→GA\rightarrow B\rightarrow D\rightarrow E\rightarrow C\rightarrow F\rightarrow GA→B→D→E→C→F→G

    中-左-右

  2. 中序遍历：D→B→E→A→F→C→GD\rightarrow B\rightarrow E\rightarrow A\rightarrow F\rightarrow C\rightarrow GD→B→E→A→F→C→G

    左-中-右

  3. 后续遍历：D→E→B→F→G→C→AD\rightarrow E\rightarrow B\rightarrow F\rightarrow G\rightarrow C\rightarrow AD→E→B→F→G→C→A

    左-右-中
